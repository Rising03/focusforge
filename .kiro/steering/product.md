# Product Overview

## Student Discipline System

A production-grade discipline and daily focus web application designed for students to build consistency, track progress, and optimize their daily routines.

## Core Features

- **Daily Routines**: AI-powered personalized schedules based on goals, energy patterns, and available time
- **Habit Tracking**: Build consistency with intelligent habit stacking, streak tracking, and completion logging
- **Progress Analytics**: Detailed insights into patterns, consistency scores, and behavioral trends
- **Focus Mode**: Distraction-free environment for deep work sessions with attention training
- **Activity Tracking**: Monitor time utilization, focus quality, and session durations
- **Evening Reviews**: Structured reflection and planning for continuous improvement
- **Identity-Based Habits**: Align habits with target identity and long-term goals
- **AI Integration**: Smart suggestions, routine optimization, and natural language input (Google Gemini API)
- **Data Export**: Export user data for mobile apps and external analysis
- **Break Suggestions**: Low-stimulation break recommendations to maintain focus

## User Experience

- Mobile-responsive design that works seamlessly across all devices
- PWA support for installation as a native app
- Google OAuth integration for easy authentication
- Real-time feedback and progress visualization
- Behavioral pattern analysis for personalized recommendations

## Target Users

Students seeking to build discipline, improve focus, and establish consistent daily routines for academic and personal success.
